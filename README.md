# Calibrate Camera Daheng MER2-2000-19U3C

โปรเจคนี้เป็นโปรแกรมสำหรับ calibrate กล้อง Daheng MER2-2000-19U3C โดยใช้ Python

## การติดตั้ง

1. Clone repository นี้:
   ```
   git clone <repository-url>
   cd Calibate_camera_daheng_MER2_2000_19U3C
   ```

2. สร้าง virtual environment:
   ```
   python -m venv venv
   source venv/bin/activate  # บน Windows: venv\Scripts\activate
   ```

3. ติดตั้ง dependencies:
   ```
   pip install -r requirements.txt
   ```

## การใช้งาน

รันโปรแกรมหลัก:
```
python main.py
```

## ความต้องการ

- Python 3.8+
- กล้อง Daheng MER2-2000-19U3C
- Libraries ที่จำเป็น (ดูใน requirements.txt)
